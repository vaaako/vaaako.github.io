Nessa página eu vou te contar um pouquinho de mim

Eu sou eu, só fiz esse site porque gosto de anotar as coisas que sei, tenho medo de algum dia de repente esquecer algo de importante que penso ou sei, como uma amnsesia repentina, por isso sempre escrevo e catalogo as coisa que sei. Eu também gosto de escrever coisas pra mim mesmo, então cá estamos.

Também queria um lugar pra colocar isso tudo e mostrar para os outros.

Não estou escrevendo essas coisas para alguém ver, escrevo apenas como uma terapia ou algo assim, gosto de escrever coisas como se alguém fossem lê-las. Escrever para mim mesmo.

Eu já tentei criar um site nesse estilo algumas vezes, mas sempre acaba comigo usando Flask ou Express pra fazer alguma coisa mais complexa do que queria no inicio, desssa vez vou tentar usar apenas HTML, CSS e Javascript (Embora meus conhecimentos pra web não sejam tão vastos, na verdade não são vastos pra nada), eu não gosto de programação web, mas vida que segue.

Explicação muito longa só pra falar que eu quis fazer esse site simples.
